---
打开此选项后，屏幕上方会有一排叠加显示的图层


| 参数    | 含义     |
| :------------- | :------------- |
| p:0/1       | 第一格显示的是触控点球（几点触控）       |
| dX:       | 第二格是当前横坐标（点击结束后会显示和上一次点的横轴差值）       |
| dY:       | 第三格是当前纵坐标（点击结束后会显示和上一次点的纵轴差值）      |
| Xv:       | 第四格是横轴位移速度       |
| Yv:       | 第五格是纵轴位移速度       |
| Prs:       | 第六格是压力值       |
| Size:       | 第七格是与屏幕接触面积       |
